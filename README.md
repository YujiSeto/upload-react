# Upload React

This is a simple Next.js application that allows users to upload an image with a legend.

## Features

- **File Upload:** Users can select a file from their local machine or drag and drop it into the application.
- **Image Preview:** A preview of the selected image is displayed before uploading.
- **Upload Progress:** A progress bar shows the upload progress.
- **Dark/Light Theme:** The application supports both dark and light themes.

## Getting Started

First, clone the repository and install the dependencies:

```bash
git clone https://github.com/YujiSeto/upload-react.git
cd upload-react
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
